# 百マス計算オンライン (100-cell Calculation Online)

ブラウザで手軽に遊べる・印刷できる、美しく高機能な「百マス計算（100マス計算）」Webアプリケーションです。

## 主な機能

1. **多彩な演算モードとサイズ**
   - 足し算（+）、引き算（-）、掛け算（×）に対応
   - グリッドサイズを標準の「10 × 10 (100マス)」と短時間用の「5 × 5 (25マス)」から選択可能
   - 引き算では「答えがマイナスにならない」自動調整オプションを提供
   - 難易度調整（かんたん、ふつう、むずかしい）に対応

2. **タイムアタックと自動採点**
   - ミリ秒単位のストップウォッチ機能
   - 進捗状況を示すリアルタイムプログレスバー
   - リアルタイム正誤判定モードと、一括採点モードの切り替え
   - 全問正解時のコンフェッティ（紙吹雪）によるお祝いアニメーション

3. **快適な操作性**
   - キーボードの矢印キー（`↑``↓``←``→`）でのグリッド移動
   - `Enter` または `Tab` キーによる自動カーソル送り機能（右に進む・下に進むの選択が可能）
   - タブレットやスマートフォンでの入力に対応する、スライド式の仮想テンキーを内蔵
   - Web Audio API を活用した、アプリ単体で動作する合成効果音（クリック音、正解音、不正解音、ファンファーレ）

4. **学習履歴と成績分析**
   - `localStorage` を使用し、ブラウザを閉じても過去のスコア（タイム、正答率、ミス数）を保存
   - 同じ演算モードでのタイム推移をグラフ（動的SVGラインチャート）で視覚化

5. **A4印刷対応のプリント作成機能 (プリントモード)**
   - 学校の宿題や自宅学習用に、「問題用紙」および「解答付きの答え合わせ用紙」をA4サイズに美しくフィットさせて印刷可能
   - 印刷時にはナビゲーションやコントロールボタン、不要な背景などの要素は自動的に非表示になり、白黒高コントラストのグリッドのみが印刷されます

6. **プレミアムデザインテーマ**
   - 深みのある「ダークモード」（デフォルト）
   - 明るくスッキリした「ライトモード」
   - ネオンとサイバーカラーが躍動する「サイバーパンクテーマ」

---

## 使い方・起動方法

このプロジェクトは、外部ライブラリ（React, Vue, Tailwind CSSなど）を一切使用せず、純粋なHTML/CSS/JS（バニラ）だけで作られています。

### 1. ローカルで開く
任意のブラウザで `index.html` ファイルを直接開くだけで起動します。
```bash
# macOS のターミナルから直接開く場合
open index.html
```

### 2. 印刷する
1. 画面の「プリント・印刷」セクションで「問題用紙を印刷」または「解答付きで印刷」をクリックします。
2. ブラウザの印刷ダイアログが開くため、レイアウトを「縦」、用紙サイズを「A4」に設定し、印刷またはPDFとして保存します。
